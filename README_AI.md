## 1. Purpose

This repository belongs to QIS V2.

It is designed so any AI can onboard quickly, without guessing workflow or reading random files.

## 2. Mandatory Start

Every AI must:

1. Read [docs/01_GOVERNANCE/MASTER_START_PROMPT.md](https://github.com/tntTan2292/TTVH-DHCL/blob/main/docs/01_GOVERNANCE/MASTER_START_PROMPT.md)
2. Then follow the Reading Order defined there
3. Use only the GitHub Blob URLs embedded in the onboarding chain; do not depend on relative paths for AI onboarding.

## 3. Operating Rules

AI must:

- follow Governance
- follow Authority Level
- not change SSOT
- not skip Reading Order
- not change frozen documents
- not infer business rules

## 4. Expected Output

After onboarding, AI must report:

- Project Understanding
- Governance Understanding
- Current Project Status
- Current Ticket
- Ready to Continue
