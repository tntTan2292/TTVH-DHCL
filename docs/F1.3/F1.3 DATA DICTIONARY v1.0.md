# 15. SOURCE FILE COLUMN MAPPING

## Mục tiêu

Ánh xạ dữ liệu từ file Excel Tổng công ty sang Database.

Nguyên tắc:
- Một cột Excel chỉ ánh xạ tới một field hệ thống.
- Không được suy diễn dữ liệu.
- Mọi thay đổi phải cập nhật Data Dictionary trước khi code.

---

## 15.1 Import Strategy & Field Selection

- **Lưu trữ toàn vẹn:** Hệ thống lưu toàn bộ 41 cột từ file Excel TCT vào Database. Không loại bỏ các cột chưa dùng.
- **Phạm vi hiển thị MVP:** Dashboard MVP chỉ sử dụng một phần dữ liệu (các trường định nghĩa trong bảng dưới đây).

## 15.2 Trường dữ liệu Hệ thống (System Fields)

| Field | Nguồn | Ví dụ | Mục đích |
|-------|-------|-------|----------|
| `ngay_do_kiem` | Tên file | `F1.3-2026.06.18.xlsx` => `2026-06-18` | Hỗ trợ phân tích: Theo ngày, 7 ngày, 10 ngày, Tháng, Quý, Năm |
